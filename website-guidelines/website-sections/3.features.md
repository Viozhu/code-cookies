# Features Specifications

## Content
Three main features to highlight:
1. **Logic Puzzles:** Visual breakdown of logic-based puzzle system
2. **Cookie Reward System:** How users earn cookies (1-5 per task)
3. **Python Bridge:** Seamless transition from puzzles to Python coding

Each feature should include:
- Icon or illustration
- Title
- Description
- Visual example or mockup

## Animations / Effects
### Feature Card Hover
- Type: [Framer Motion]
- Reference: Card lifts slightly on hover with shadow increase
- Code:
```tsx
<motion.div
  whileHover={{ y: -5 }}
  transition={{ duration: 0.2 }}
>
  <FeatureCard />
</motion.div>
```

### Cookie Count Animation
- Type: [Framer Motion]
- Reference: Cookies animate in sequentially when feature is viewed
- Code:
```tsx
{cookies.map((cookie, index) => (
  <motion.div
    key={index}
    initial={{ opacity: 0, scale: 0 }}
    animate={{ opacity: 1, scale: 1 }}
    transition={{ delay: index * 0.1 }}
  >
    <CookieIcon />
  </motion.div>
))}
```

### Icon Pulse Animation
- Type: [CSS / Framer Motion]
- Reference: Feature icons have subtle pulse animation
- Code:
```tsx
<motion.div
  animate={{ scale: [1, 1.1, 1] }}
  transition={{ duration: 2, repeat: Infinity }}
>
  <FeatureIcon />
</motion.div>
```

## Design Instructions
- **Spacing:** Equal spacing between cards (24-32px gap)
- **Card Height:** Consistent card heights for visual balance
- **Icons:** Large, colorful icons (64px minimum) that are easily recognizable
- **Visual Examples:** Include interactive previews or high-quality mockups
- **Color Scheme:** Each feature can have its own accent color
- **Responsive:** Stack cards vertically on mobile, 2 columns on tablet
- **Accessibility:** High contrast, clear labels, descriptive alt text for visuals
