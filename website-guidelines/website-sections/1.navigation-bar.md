# Navigation Bar Specifications

## Content
- **Logo:** CodeCookies brand logo/icon
- **Navigation Links:**
  - "My Progress" - Link to user's progress dashboard
  - "Cookie Jar" - Link to rewards/achievements page showing earned cookies
- **User State:** Should handle both logged-in and logged-out states

## Animations / Effects
### Cookie Jar Badge Animation
- Type: [Framer Motion]
- Reference: Animated cookie count badge that pulses when cookies are earned
- Code:
```tsx
<motion.div
  animate={{ scale: [1, 1.2, 1] }}
  transition={{ duration: 0.3 }}
  className="relative"
>
  <CookieIcon />
  <Badge className="absolute -top-2 -right-2">{cookieCount}</Badge>

```

## Design Instructions
- **Spacing:** Padding of 12-16px vertical, comfortable horizontal spacing between elements
- **Height:** Minimum 64px for touch-friendly targets
- **Responsive:** Stack vertically on mobile with hamburger menu
- **Accessibility:** Large clickable areas (minimum 44x44px), high contrast text
- **Behavior:** Fixed position, stays visible on scroll
- **Cookie Jar Link:** Should display current cookie count if user is logged in
