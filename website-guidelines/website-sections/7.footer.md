## Animations / Effects
### Link Hover Effect
- Type: [CSS]
- Reference: Links change color and underline on hover
- Code:
```tsx
<Link 
  href={href}
  className="hover:text-white transition-colors relative group"
>
  {text}
  <span className="absolute bottom-0 left-0 w-0 h-0.5 bg-white group-hover:w-full transition-all duration-300" />
</Link>
```

### Social Icon Hover
- Type: [Framer Motion]
- Reference: Social icons scale and glow on hover
- Code:
```tsx
<motion.a
  href={url}
  whileHover={{ scale: 1.2 }}
  whileTap={{ scale: 0.9 }}
  className="w-8 h-8 rounded-full bg-gray-800 flex items-center justify-center hover:bg-gray-700 transition-colors"
>
  <SocialIcon />
</motion.a>
```

### Fade In Animation
- Type: [Framer Motion]
- Reference: Footer sections fade in on scroll
- Code:
```tsx
<motion.div
  initial={{ opacity: 0, y: 20 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true }}
  transition={{ duration: 0.5 }}
>
  <FooterSection />
</motion.div>
```

## Design Instructions
- **Background:** Dark background (gray-900 or black) for contrast with main content
- **Layout:** 4-column grid on desktop, stacked on mobile
- **Spacing:** Generous padding (48px vertical), comfortable gaps between sections
- **Typography:** Smaller text sizes (14px body), clear hierarchy
- **Links:** All links should be easily clickable with hover states
- **Legal Compliance:** Prominent display of COPPA/GDPR compliance badges/links
- **Responsive:** Stack sections vertically on mobile
- **Accessibility:** High contrast text, clear focus states, semantic HTML
- **Contact Info:** Email should be clickable mailto link
- **Social Icons:** Optional but recommended for brand presence
