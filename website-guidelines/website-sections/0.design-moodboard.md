# Design Moodboard Specifications

## Overall Design Direction
- **Style:** Modern, playful, child-friendly educational platform
- **Mood:** Energetic, approachable, fun, trustworthy
- **Key Principles:** High contrast, large touch targets, vibrant colors, clear visual hierarchy

## Color Palette

### Primary Colors
- **Primary:** `#4B19AE` - Main brand color, CTAs, primary actions
- **Secondary:** `#C4FF3E` - Accent color, highlights, secondary actions

### Primary Variations
- **Primary Light:** `#6B3BC7` - Hover states, lighter backgrounds
- **Primary Dark:** `#3A1385` - Darker shade for contrast
- **Primary Subtle:** `#E8DFF5` - Very light tint for backgrounds

### Secondary Variations
- **Secondary Light:** `#D4FF6B` - Lighter tint for backgrounds
- **Secondary Dark:** `#A8E030` - Darker shade for text/contrast
- **Secondary Subtle:** `#F5FFE0` - Very light tint for backgrounds

### Neutrals
- **Background Light:** `#FFFFFF` (White)
- **Background Dark:** `#0F0A1F` (Dark purple-tinted)
- **Text Light:** `#1F1F1F` (Near black)
- **Text Dark:** `#F9FAFB` (Light gray)
- **Border Light:** `#E5E7EB` (Gray-200)
- **Border Dark:** `#374151` (Gray-700)

### Gradients
- **Hero Gradient:** `from-[#4B19AE] via-[#6B3BC7] to-[#C4FF3E]`
- **CTA Gradient:** `from-[#4B19AE] to-[#6B3BC7]`
- **Accent Gradient:** `from-[#C4FF3E] to-[#D4FF6B]`

## Typography

### Type Scale
- **H1 (Hero):** 48px-64px - Bold, gradient text
- **H2 (Section Headers):** 36px-48px - Bold
- **H3 (Subsection):** 24px-30px - Semi-bold
- **Body:** 16px - Regular
- **Small:** 14px - Regular

## Layout & Spacing

### Grid System
- **Container:** Max-width 1280px, centered
- **Padding:** 16px-24px mobile, 24px-48px desktop
- **Spacing:** 4px base unit (4, 8, 12, 16, 24, 32, 48, 64)

### Component Styles
- **Cards:** Rounded corners (12px-16px), subtle shadows
- **Buttons:** Large (min 44px height), rounded (8px-12px)
- **Touch Targets:** Minimum 44x44px

## Animation Principles
- **Style:** Playful, bouncy, not distracting
- **Duration:** 150-300ms for interactions, 500-800ms for transitions
- **Key Animations:** Cookie earned bounce, button hover scale, page fade transitions

## Responsive Breakpoints
- **Mobile:** 320px-768px
- **Tablet:** 768px-1024px
- **Desktop:** 1024px+
